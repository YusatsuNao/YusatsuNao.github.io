Anime Icon

Anime Icon is a Folder Icon with Anime Image, you can use it to change every anime folder's icon you have. So, if you want to watch the anime again and forgot the title you can know where is it by looking for the image or just make the folder looks cool. Tab below is the list of the Anime Icon I make, and only available for Windows only but I will make for Macintosh too but that takes a time.

For folder icon, I'm using a .dll files, but I will post the .ico and .png files too if you want. For more information about the Icon you can see the Download Tab below and click This Post. If you want request an Anime Icon you can check About Page then find QSR ( Question, Suggestion or Request ) part and click Google Forms link also if you want new version of icons please request it from there so I can make it for you and post it here and on my DeviantArt page.

1. What is .dll file?

.dll or DLL file is a dynamic link library file format used for holding multiple codes and procedures for Windows programs but this file can also store a other Windows file like image, small programs or etc for larger program to running.
I use .dll file to store the Anime Icon in one pack. So you can easily choose the icon.

2. How I can open it? the .dll?

You don't need to open it, if you want to change your folder icons, just right clicking your folder then a window appear, choose Customize tab then click Change Icon... button, browser where the .dll file, after you choose and click the .dll file all of the Anime Icons will shown up. Choose one that you like. click ok, ok again. Finish.

3. How I can change the folder icon??

Same as above.

4. DLL or ICO?

Just choose one, .ico file or .dll files. THEY ARE SAME.
